I worked on this system when I was employed in Adabisc future qatar. We wanted to make real-time face tracking system for one of our products, and I decided to build this system with Intel real-sense camera SR300 which had IR sensor and decent depth information. I had to build a face tracking system using neural networks so that it would track a certian face in certain poses, then I just use that training data to map between face tracking points and blend shapes on the final model.

So I started by trying to make a plugin for Unity game engine from Intel real-sense sdk and it was kind of tricky specially because they did provide us with working plugins. After that, I had to search for a suitable learning algorithm to convert the face markers positions to blendshape weights on the target model. So I found out that the best machine learning algorithm to use was radial base function. So first, I had to create a data set.

Collecting data and training stage:

So in the collecting data stage, we ask the subject to perform multiple poses in front of the camera and during that, we collect his face tracking points positions and save them to a file on the desk. Then we feed the learning algorithm the data and let it train its variables. This process takes from 30 sec to 1 minute.

Face Tracking.

Now comes the fun part where I just use the trained variables to do the mapping process.
