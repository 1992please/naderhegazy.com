I worked on that system when I employ in adabisc future qatar. we wanted to make real time face tracking system for on of our products. and I decided to build that system with intel real sense camera SR300 which had IR sensor and decent depth information. I had to build a face tracking system using nueral networks so it will train on certian face in certain poses and then I just use that training data to map between face tracking points and blend shapes on the final model.

so I started by trying to make plugin for unity game engine from intel real sense sdk and it was kinda tricking specially because they did provide us with working plugins. after that I had to search for suitable learning algorithm to convert the face markers positions to blendshape sweigts on the target model. so I found out the best machine learning algorithm to use was radial base function. so first I had to create a data set.

Collecting data and training stage:

so in the collecting data stage, we ask the subject to perform multiple poses to the camera before and during that we collect his face tracking points positions and save them to a file on desk. then we feed the learning algorithm the data and let it train it's variables. this process take like 30 sec to 1 minute.

Face Tracking.

now comes the fun part where I just use the trained variables to do the mapping process.
