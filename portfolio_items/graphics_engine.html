A personal graphics engine to learn low level graphics programing.

Fbx_sdk

First, all I needed was to be able to import just the vertices from the Fbx file and I added any texture to the material just for test.

Then I wanted to be able to import the materials with textures from the Fbx-SDK.

Testing Stencil

Next I had to reorganize the project code to be able to add more feature, more shaders and more flexibility

Instantiation Implementation.

Instantiation is one the of the most important aspects of any graphical engine, and luckily for me OpenGl have some features that makes that easier for me. So after a lot of trial and error I was able to instantiate 100,000 rocks around a planet without having trouble with CPU bottlenecking my system.

Skybox with reflection/refraction

Skybox is just a CubeMap that you use as a background for you scene, and then I used to it to create refraction and reflection effects. you can see in the video two cubes the right one is simulating mirror and left one is glass.
